# Notas relevantes del lenguaje

Un programa en go siempre empieza con un paquete de **main** y una funcion **main()**.

Ejemplo del archivo main.go

```go
package main

import "fmt"

func main() {
    fmt.Println("Hola, mundo")
}
```
